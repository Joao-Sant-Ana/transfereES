# Roadmap de Evolução

## Curto prazo (1-2 sprints)
- Implementar autenticação com sessão e controle de acesso por perfil.
- Criar API routes para CRUD de projeções, pressões e validações.
- Adicionar importação real de CSV/XLSX com validação de colunas.
- Implementar exportação inicial de relatório executivo em Excel.

## Médio prazo (3-5 sprints)
- Consolidar cálculo oficial do painel com persistência por data de referência.
- Implementar relatório técnico em PDF com assinaturas e trilha de alterações.
- Criar central de alertas e notificação interna por status de farol.

## Longo prazo
- Integração automatizada com SIGEFES por API.
- Painel de tendências e projeções comparativas entre competências.
- Governança de dados com trilha de homologação multi-etapas.
